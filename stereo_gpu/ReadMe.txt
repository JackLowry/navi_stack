
* Notes:

1) Usage sample is provided in test directory.
2) disp_format CV_8U, 0 - for filtered values. This is in contrast with OpenCV's CPU verstion, where CV_16S and -1.
  Mat::convertTo on GPU is not included in this package.


* To compile this package please:

Install Cuda Toolkit 3.0 with drivers for it.
http://developer.nvidia.com/object/cuda_3_0_downloads.html

or 

Install Cuda Toolkit 3.1 with drivers for it.
http://developer.nvidia.com/object/cuda_3_1_downloads.html


or 

Install Cuda Toolkit 3.2 with drivers for it.
http://developer.nvidia.com/object/cuda_3_2_toolkit_rc.html
Update  FindCUDA.cmake script in CMake modules directory to support this version
